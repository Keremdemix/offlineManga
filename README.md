# 📚 Offline Manga App

Offline Manga App is a mobile application that allows users to read manga completely offline.  
Manga chapters are stored locally on the device, providing a fast, smooth, and uninterrupted reading experience without internet dependency.

---

## 🚀 Features

- 📖 Offline manga reading support
- ⚡ Fast and smooth chapter loading
- 📂 Local storage for downloaded manga
- 🔖 Continue reading from last position
- 🎨 Clean and minimal UI optimized for reading
- 📱 Mobile-friendly design

---

## 🧠 Project Purpose

This project was developed to improve user experience in manga reading by eliminating internet dependency and focusing on performance, simplicity, and accessibility.

The goal is to provide a distraction-free reading experience with efficient local data handling.

---

## 🛠️ Tech Stack

- React Native
- JavaScript / TypeScript
- Android Studio
- React Navigation
- AsyncStorage / Local Storage
- MVVM-inspired architecture
- REST API Integration (optional)

---


## 🏗️ Architecture

The project follows MVVM architecture:

- **UI Layer (Compose)** → User interface
- **ViewModel Layer** → State management
- **Repository Layer** → Data handling
- **Local Database (Room)** → Offline storage
